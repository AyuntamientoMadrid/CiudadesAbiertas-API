4 Formularios

A continuación se muestran los diferentes componentes para construir formularios.

4.1 Input

Se utilizan los input personalizados de Bootstrap:

hay tres tipos de input

  • input tipo texto type="text"
  • input tipo numérico type="number"
  • input tipo archivo type="file"
<div class="form-group">
    <label class="control-label">Campo texto</label>
    <input type="text" class="form-control" name="username" required="true">
</div>
<div class="form-group">
    <label class="control-label">Campo numérico</label>
    <input type="number" class="form-control" name="username">
</div>
<div class="form-group">
    <input type="file" class="custom-file-input" id="inputGroupFile01">
    <label class="custom-file-label label_show" for="inputGroupFile01">Seleccione un archivo</label>
    <label class="custom-file-label label_hidden hidden" for="inputGroupFile01"></label>
    <i class="custom-file-icon icon icon-close hidden" alt="borrar" title="borrar"></i>
</div>

4.2 Select

Los elementos select tienen una presentación especifica. Su personalización se hace por javascript, por lo que basta con incluirlos como un select normal.
Se utilizan si desea permitir que el usuario elija entre varias opciones

<div class="form-group">
<label class="control-label">Lista de valores (desplegable)</label>
<div class="custom-select">
<select class="form-control simple" name="desplegable">
    <option value="">Seleccione una opción</option>
    <option value="op1">Opción 1</option>
    <option value="op2">Opción 2</option>
    <option value="op3">Opción 3</option>
</select>
<span class="custom-select__arrow" aria-hidden="true">
<span class="icon-arrow-full-down"></span>
</span>
</div>
</div>

4.3 Checkbox

Corresponden a los checkbox personalizados de Bootstrap y se utilizan para seleccionar una o varias opciones en una lista

<div class="form-check">
    <input class="form-check-input" type="checkbox" value="" id="defaultCheck1">
    <label class="form-check-label" for="defaultCheck1">
    Checkbox
    </label>
</div>

4.4 Radio

Corresponden a los radio personalizados de Bootstrap y se utilizan para seleccionar una opción entre muchas.

<div class="form-check">
    <input class="form-check-input" type="radio" name="exampleRadios" id="exampleRadios1" value="option1" checked="">
    <label class="form-check-label" for="exampleRadios1">
    Radiobutton
    </label>
</div>

4.5 Campo texto (Bloque)

Es un campo de texto dedicado para un gran volumen de texto.
Se utilizan los textarea personalizados de Bootstrap.

<div class="form-group">
    <label for="exampleFormControlTextarea1">Campo texto (Bloque)</label>
    <textarea class="form-control" id="exampleFormControlTextarea1" rows="3"></textarea>
</div>

4.6 Campo fecha

Se utilizan los input personalizados de Bootstrap a los que hay que incluir las clases from hasDatepicker.

<div class="form-group">
    <div class="col-sm-12 col">
        <label class="control-label">Campo fecha</label>
        <input type="text" id="date" class="from hasDatepicker" name="date" placeholder="Campo fecha">
        <img class="ui-datepicker-trigger" src="img/calendar.png" alt="..." title="...">
    </div>
</div>

4.7 Casilla de verificación

Se utiliza la casilla de verificación personalizados de Bootstrap.

Casilla de verificación 
<div class="form-group">
    <label class="switch">
        <input type="checkbox">
        <span class="slider round"></span>
    </label>
    <span class="texto_switch">Casilla de verificación</span>
</div>

4.8 Validación de campos

Cuando un campo del formulario es obligatorio * tiene una validación, estos campos tienen dos estados:

  • Correcto class="okField"
  • Incorrecto class="errorField"
Texto cuando hay un 'error'. 
<div class="form-group">
    <label class="control-label">Campo texto error *</label>
    <input type="text" class="form-control errorField" name="username" required="true">
    <span class="hint">Texto cuando hay un <strong>'error'</strong>. </span>
</div>
<div class="form-group">
    <label class="control-label">Campo texto ok *</label>
    <input type="email" class="form-control okField" name="username" required="true">
</div>
