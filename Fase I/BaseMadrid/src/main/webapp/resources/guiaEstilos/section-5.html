5 Botones

Los estilos de botón se pueden aplicar a cualquier elemento con la clase .btn aplicada. Sin embargo, normalmente se aplica solo a los elementos <a> y <button> para obtener la mejor representación.

5.1 Estilos

Hay tres tipos de botones:

  • Botón verde: btn-success
  • Botón gris: btn-secondary
  • Botón rojo: btn-danger
<button type="submit" class="btn btn-success">Button</button>
<button type="submit" class="btn btn-secondary">Button</button>
<button type="submit" class="btn btn-danger">Button</button>

5.2 Estados

:hover

Botón hover.

.focus

Botón activo.

.disabled

Botón desactivado. Si usa el elemento <button>, se le puede agregar el atributo disabled en su lugar.

<button type="submit" class="btn btn-success {$modifiers}">Button</button>
<button type="submit" class="btn btn-secondary {$modifiers}">Button</button>
<button type="submit" class="btn btn-danger {$modifiers}">Button</button>
