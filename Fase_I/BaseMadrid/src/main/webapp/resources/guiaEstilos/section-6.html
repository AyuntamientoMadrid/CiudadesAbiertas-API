6.2 Media Queries

Los anchos para las media queries por dispositivo son:

  • Móvil: Hasta 767px ($screen-xs-max)
  • Tablet: Desde 768px ($screen-md-min) hasta 1023px ($screen-md-max)
  • Escritorio: A partir de 1024px ($screen-lg-min)

Trabajamos con una aproximación Mobile First, por lo que generalmente daremos estilos para móvil y usaremos estas media queries para tablet y escritorio:

@media (min-width: $screen-md-min) {
    CSS Tablet
}

@media (min-width: $screen-lg-min) {
    CSS Escritorio
}
  • $screen-xl-min: 1200px;
  • $screen-lg-max: 1199px;
  • $screen-lg-min: 1024px;
  • $screen-md-max: 1023px;
  • $screen-md-min: 768px;
  • $screen-xs-max: 767px;

6.3 Utilidades

Funciones SASS.

em

em($target, $context: $font-base)

Convierte una medida de px en em.
$target: Los px objetivo.
$context: Los px del contexto. Este parámetro es opcional, si no se indica se tomarán 16px ($font-base).
return: Tamaño en em.
