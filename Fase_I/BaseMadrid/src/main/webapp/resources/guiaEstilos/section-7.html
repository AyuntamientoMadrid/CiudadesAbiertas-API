7 Aplicaciones Externas

En esta sección se incluyen una serie de pautas recomendadas para las aplicaciones externas, que junto a las descritas en los apartados anteriores, permite construir aplicaciones con el nuevo modelo de presentación de la Intranet.
Para la creación de los interfaces de las aplicaciones se ha confeccionado una estructura común simplificada.

7.1 Cabecera

Para las aplicaciones externas, la cabecera no contiene buscador ni menú principal y dispone de sus opciones locales.

7.3 Estructura

La estructura es la misma que en la intranet del ayuntamiento que se describe en el punto 1.1 de esta quía, el desarrollador deberá tener en cuenta esto a la hora de crear las diferentes pantallas.
Dentro de esta estructura se pueden incluir los elementos comunes descritos en el apartado 3 de esta guía.
